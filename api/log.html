log¶

日志库

log.setLevel(level)¶

设置日志级别

参数¶

传入值类型

解释

string

level 日志级别,可用字符串或数值, 字符串为(SIL ENT,DEBUG,INFO,WARN,ERROR,FATAL), 数值为(0,1,2,3,4,5)

返回值¶

返回值类型

解释

string

level 日志级别,可用字符串或数值, 字符串为(SIL ENT,DEBUG,INFO,WARN,ERROR,FATAL), 数值为(0,1,2,3,4,5)

返回值¶

-- 设置日志级别为INFO
log.setLevel("INFO")
