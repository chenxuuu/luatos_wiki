rtos¶

RTOS底层操作库

rtos.receive(timeout)¶

接受并处理底层消息队列.

参数¶

传入值类型

解释

int

超时时长,通常是-1,永久等待

返回值¶

返回值类型

解释

int

超时时长,通常是-1,永久等待

返回值¶

无

rtos.timer_start(id,timeout,_repeat)¶

启动一个定时器

参数¶

传入值类型

解释

int

定时器id

int

超时时长,单位毫秒

int

重复次数,默认是0

返回值¶

返回值类型

解释

int

定时器id

int

超时时长,单位毫秒

int

重复次数,默认是0

返回值¶

-- 用户代码请使用 sys.timerStart
-- 启动一个3秒的循环定时器
rtos.timer_start(10000, 3000, -1)

rtos.timer_stop(id)¶

参数¶

传入值类型

解释

int

定时器id

返回值¶

返回值类型

解释

int

定时器id

返回值¶

-- 用户代码请使用sys.timerStop
rtos.timer_stop(100000)

rtos.meminfo(type)¶

获取内存信息

参数¶

传入值类型

解释

type

“sys”系统内存, “lua”虚拟机内存, 默认为lua虚拟机内存

返回值¶

返回值类型

解释

type

“sys”系统内存, “lua”虚拟机内存, 默认为lua虚拟机内存

返回值¶

-- 打印内存占用
log.info("mem.lua", rtos.meminfo())
log.info("mem.sys", rtos.meminfo("sys"))

参数¶

无

返回值¶

返回值类型

解释

返回值¶
