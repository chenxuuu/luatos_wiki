libcoap¶

coap数据处理

libcoap.new(code, uri, headers, payload)¶

创建一个coap数据包

参数¶

传入值类型

解释

int

coap的code, 例 如libcoap.GET/libcoap.POST/libcoap.PUT/libcoap.DELETE

string

目标URI,必须填写, 不需要加上/开头

table

请求头,类似于http的headers,可选

string

请求体,类似于http的body,可选

返回值¶

返回值类型

解释

int

coap的code, 例 如libcoap.GET/libcoap.POST/libcoap.PUT/libcoap.DELETE

string

目标URI,必须填写, 不需要加上/开头

table

请求头,类似于http的headers,可选

string

请求体,类似于http的body,可选

返回值¶

-- 创建一个请求服务器time的数据包
local coapdata = libcoap.new(libcoap.GET, "time")
local data = coapdata:rawdata()
