i2c
===

I2C操作

i2c.exist(id)
-------------

i2c编号是否存在

参数
~~~~

========== ====================================
传入值类型 解释
========== ====================================
int        设备id, 例如i2c1的id为1, i2c2的id为2
========== ====================================

返回值
~~~~~~

========== ====================================
返回值类型 解释
========== ====================================
int        设备id, 例如i2c1的id为1, i2c2的id为2
========== ====================================

.. _返回值-1:

返回值
~~~~~~

.. code:: lua

   -- 检查i2c1是否存在
   if i2c.exist(1) then
       log.info("存在 i2c1")
   end

i2c.readDHT12(id)
-----------------

从i2c总线读取DHT12的温湿度数据

.. _参数-7:

参数
~~~~

========== ====================================
传入值类型 解释
========== ====================================
int        设备id, 例如i2c1的id为1, i2c2的id为2
int        DHT12的设备地址,默认0x5C
========== ====================================

.. _返回值-14:

返回值
~~~~~~

========== ====================================
返回值类型 解释
========== ====================================
int        设备id, 例如i2c1的id为1, i2c2的id为2
int        DHT12的设备地址,默认0x5C
========== ====================================

.. _返回值-15:

返回值
~~~~~~

.. code:: lua

   -- 从i2c0读取DHT12
   i2c.setup(0)
   local re, H, T = i2c.readDHT12(0)
   if re then
       log.info("dht12", H, T)
   end
