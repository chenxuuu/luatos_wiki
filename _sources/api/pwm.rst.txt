pwm
===

PWM模块

pwm.open(channel, period, pulse)
--------------------------------

开启指定的PWM通道

参数
~~~~

========== =================
传入值类型 解释
========== =================
int        PWM通道
int        频率, 1-1000000hz
int        占空比 0-100
========== =================

返回值
~~~~~~

========== =================
返回值类型 解释
========== =================
int        PWM通道
int        频率, 1-1000000hz
int        占空比 0-100
========== =================

.. _返回值-1:

返回值
~~~~~~

.. code:: lua

   -- 打开PWM5, 频率1kHz, 占空比50%
   pwm.open(5, 1000, 50)

--------------
