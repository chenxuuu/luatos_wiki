eink
====

墨水屏操作库

eink.setup(full, spiid)
-----------------------

初始化eink

参数
~~~~

========== ==================================
传入值类型 解释
========== ==================================
int        全屏刷新1,局部刷新0,默认是全屏刷新
int        所在的spi,默认是0
========== ==================================

返回值
~~~~~~

========== ==================================
返回值类型 解释
========== ==================================
int        全屏刷新1,局部刷新0,默认是全屏刷新
int        所在的spi,默认是0
========== ==================================

.. _返回值-1:

返回值
~~~~~~

.. code:: lua

   无
