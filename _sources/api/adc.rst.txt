adc
===

数模转换

adc.open(id)
------------

打开adc通道

参数
~~~~

========== =================================
传入值类型 解释
========== =================================
int        通道id,与具体设备有关,通常从0开始
========== =================================

返回值
~~~~~~

========== =================================
返回值类型 解释
========== =================================
int        通道id,与具体设备有关,通常从0开始
========== =================================

.. _返回值-1:

返回值
~~~~~~

.. code:: lua

   -- 打开adc通道2,并读取
   if adc.open(2) then
       log.info("adc", adc.read(2))
   end
   adc.close(2)

--------------
