lpmem
=====

操作低功耗不掉电内存块

lpmem.read(offset, size)
------------------------

读取内存块

参数
~~~~

========== =================
传入值类型 解释
========== =================
int        内存偏移量
int        读取大小,单位字节
========== =================

返回值
~~~~~~

========== =================
返回值类型 解释
========== =================
int        内存偏移量
int        读取大小,单位字节
========== =================

.. _返回值-1:

返回值
~~~~~~

.. code:: lua

   -- 读取1kb的内存
   local data = lpmem.read(0, 1024)

--------------

lpmem.write(offset, str)
------------------------

写入内存块

.. _参数-1:

参数
~~~~

========== ============
传入值类型 解释
========== ============
int        内存偏移量
string     待写入的数据
========== ============

.. _返回值-2:

返回值
~~~~~~

========== ============
返回值类型 解释
========== ============
int        内存偏移量
string     待写入的数据
========== ============

.. _返回值-3:

返回值
~~~~~~

.. code:: lua

   -- 往偏移量为512字节的位置, 写入数据
   lpmem.write(512, data)
