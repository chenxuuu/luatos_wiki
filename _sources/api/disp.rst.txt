disp
====

显示屏控制

disp.init(conf)
---------------

显示屏初始化

参数
~~~~

========== ========
传入值类型 解释
========== ========
table      配置信息
========== ========

返回值
~~~~~~

========== ========
返回值类型 解释
========== ========
table      配置信息
========== ========

.. _返回值-1:

返回值
~~~~~~

.. code:: lua

   -- 初始化i2c1的ssd1306
   if disp.init({mode="i2c_sw", pin0=17, pin1=18}) == 1 then
       log.info("disp", "disp init complete")
   end

--------------

disp.close()
------------

关闭显示屏

.. _参数-1:

参数
~~~~

无

.. _返回值-2:

返回值
~~~~~~

无

.. _返回值-3:

返回值
~~~~~~

.. code:: lua

   -- 关闭disp,再次使用disp相关API的话,需要重新初始化
   disp.close()

--------------

disp.update()
-------------

把显示数据更新到屏幕

.. _参数-2:

参数
~~~~

无

.. _返回值-4:

返回值
~~~~~~

无

.. _返回值-5:

返回值
~~~~~~

.. code:: lua

   -- 把显示数据更新到屏幕
   disp.update()

--------------

disp.drawStr(content, x, y)
---------------------------

在显示屏上画一段文字,要调用disp.update才会更新到屏幕

.. _参数-3:

参数
~~~~

========== ========
传入值类型 解释
========== ========
string     文件内容
int        横坐标
int        竖坐标
========== ========

.. _返回值-6:

返回值
~~~~~~

无

.. _返回值-7:

返回值
~~~~~~

.. code:: lua

   disp.drawStr("wifi is ready", 10, 20)

--------------

disp.setFont(fontId)
--------------------

设置字体

.. _参数-4:

参数
~~~~

+------------+--------------------------------------------------------+
| 传入值类型 | 解释                                                   |
+============+========================================================+
| int        | 字体id, 默认0,纯英文8x8字节. 如果支持中文支持,         |
|            | 那么1代表12x12的中文字体.                              |
+------------+--------------------------------------------------------+

.. _返回值-8:

返回值
~~~~~~

无

.. _返回值-9:

返回值
~~~~~~

.. code:: lua

   -- 设置为中文字体,对之后的drawStr有效
   disp.setFont(1)

--------------
