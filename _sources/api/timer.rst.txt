timer - 操作底层定时器
======================

timer.mdelay(timeout)
---------------------

硬阻塞指定时长,期间没有任何luat代码会执行,包括底层消息处理机制

**参数**

========== ========
传入值类型 解释
========== ========
int        阻塞时长
========== ========

**返回值**

========== ========
返回值类型 解释
========== ========
nil        无返回值
========== ========
